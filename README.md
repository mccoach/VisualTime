# 光阴 VisualTime

一个优雅的时间倒计时工具，让时间可视化。

## 功能特性

- 🕐 实时显示当前日期和农历
- 📅 年/季/月/周剩余时间倒计时
- ⏱️ 今日剩余时间（精确到毫秒）
- 🎯 自定义事件倒计时
- 🎨 深色主题，绿色系配色
- 💾 本地存储用户设置

## 技术栈

- Vue 3
- Vite
- Day.js

## 本地开发

    bash

    安装依赖
    npm install

    启动开发服务器
    npm run dev

    构建生产版本
    npm run build

## 部署

项目支持部署到 GitHub Pages 和 Vercel。

### GitHub Pages

1. Fork 本仓库
2. 修改 `vite.config.js` 中的 `base` 为你的仓库名
3. 推送代码到 `main` 分支
4. 在仓库设置中启用 GitHub Pages
